# Project Management System using MERN Stack
- Project Management System built using MERN (MongoDB, Express.js, React.js, Node.js) Stack. 
- That allows users to create and manage projects and tasks with a clean user interface.
- This website supports real-time task management.

## Features
- Create, update and delete projects
- Add and manage tasks within projects
- Drag and drop task movement

## Packages used
- Tailwindcss
- Headlessui
- React router
- Axios
- UUID
- Joi
- Cors
- Dotenv
